# CoinDropper
Hackathron

Simple Tetris / 2048 like game.

Move left and right with arrow keys.
Up Arrow to rotate.
Down Arrow to drop.

If 3 or more objects have the same colour/value next to each other, they will combine to form the higher tier.

Current Tiers:
1. Red
2. Green
3. Blue
